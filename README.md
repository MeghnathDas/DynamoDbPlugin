# DynamoDbPlugin
Dynamo DB plugin to access AWS dynamo DB
###### Author: Meghnath Das

### Usage
- Use configuration like this:
"AWS": {
    "Profile": "",
    "AccessKey": "",
    "SecretKey": "",
    "Region": ""
  }

### Tools & Technologies Used
- .Net Core
- AWS SDK

Hosted URL -> [https://www.nuget.org/packages/MeghnathDas](https://www.nuget.org/packages/MeghnathDas)
